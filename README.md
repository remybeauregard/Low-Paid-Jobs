# Job Preferences and Wage Expectations Among Ghanaian University Graduates

Baseline survey analysis for a CEGA research project studying wage expectations and job-choice factors among Ghanaian university graduates. Findings are intended to inform the design of a pilot RCT.

## Overview

The survey was administered via KoboCollect. Respondents reported their minimum and maximum willingness-to-accept (WTA) wages and indicated which factors influence their employment decisions (e.g. professional development, location, social influences). The cleaned analytical sample contains 48 respondents.

Key findings: professional development is positively associated with both the wage floor and ceiling; family opinion is associated with a lower wage ceiling; peer opinion is associated with a higher wage ceiling.

## Reproducing the Analysis

**Step 1 — Run the Stata script** (generates all three table files):

```bash
stata -do "Code/d1 cleaning.do"
```

Or open in the Stata GUI and run with Cmd+Shift+D (Mac) / Ctrl+D (Windows/Linux).

**Step 2 — Compile the report** (run twice to resolve cross-references):

```bash
pdflatex -output-directory Report Report/report.tex
pdflatex -output-directory Report Report/report.tex
```

The three `.tex` table files in `Report/` are auto-generated and should not be edited by hand.
